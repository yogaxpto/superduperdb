Iterates recursively through lists and dicts, yielding all leaves x

where accept(x) is True.

  • tree – A tree made of dicts and lists

  • accept – A function which returns True for values to yield

Iterates recursively through lists and dicts, rewriting all leaves x

where accept(x) is True with rewrite(x)

Parameters:

  • tree – A tree made of dicts and lists

  • accept – A function which returns True for values to rewrite

  • rewrite – A function which rewrites accepted values
